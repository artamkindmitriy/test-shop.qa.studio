# Чек-лист тестирования: Функционал магазина

## 1. Общие компоненты (Header & Footer)

### Header
- [ ] Проверить кликабельность логотипа (переход на главную).
- [ ] Проверить работу поиска (ввод текста, позитивный/негативный сценарии).
- [ ] Проверить переход в "Избранное" по клику на иконку сердечка.
- [ ] Проверить открытие мини-корзины по клику на иконку корзины.

### Навигация (Меню)
- [ ] Проверить переход в "Каталог" и отображение счетчика товаров (17/17).
- [ ] Проверить открытие/закрытие вопросов на странице "/faq".
- [ ] Проверить работу раздела "/blog" (переход по категориям, корректность счетчика постов).
- [ ] Проверить переход на страницу "/about-us".
- [ ] Проверить работу формы обратной связи на странице "/contact-us" (заполнение и отправка).
- [ ] Проверить переходы по ссылкам "Оплата и доставка" и "Политика конфиденциальности".

### Footer (Соцсети)
- [ ] Проверить корректность ссылок на внешние ресурсы:
  - [ ] VK
  - [ ] Instagram
  - [ ] Telegram
  - [ ] YouTube

## 2. Слайдер
- [ ] Проверить автоматическую прокрутку слайдов.
- [ ] Проверить ручное перелистывание слайдов (кнопки навигации/свайпы).
- [ ] Проверить соответствие изображения и текста на каждом слайде.
- [ ] Проверить работу кнопки "Купить сейчас" (переход на целевую карточку товара).

## 3. Каталог товаров

### Фильтры и категории
- [ ] Проверить переключение категорий (отображение соответствующего списка товаров).
- [ ] Проверить работу кнопки "Filter":
  - [ ] Открытие и закрытие дополнительной панели фильтров.
  - [ ] Возможность выбора параметров фильтрации.
  - [ ] Возможность сброса (очистки) примененных фильтров.
- [ ] Проверить работу кнопки "Default" (сброс к исходному).

### Карточки товаров (в списке)
- [ ] Проверить отображение элементов при наведении (появление кнопок "Избранное", "Add to cart" и "Глазок").
- [ ] Проверить добавление товара в избранное (клик по иконке сердечка).
- [ ] Проверить добавление товара в корзину (клик по "Add to cart", визуализация мини-корзины).
- [ ] Проверить открытие детальной карточки товара по клику на иконку "глаз".

### Карточки товаров (полностью)
- [ ] Проверить ввод граничных значений.
- [ ] Проверить кнопку "Add to cart" на добавление товара в мини-корзину (визуализация).
- [ ] Проверить кнопку "Добавить в избранное".
- [ ] Проверить отображение цены, описания с SKU (артикула).

## 4. Критический путь

### Мини-корзина
- [ ] Проверить ввод граничных значений количества товара.
- [ ] Проверить переходы: кнопка "View cart" -> страница корзины, кнопка "Checkout" -> оформление.

### Страница корзины
- [ ] Проверить ввод граничных значений.
- [ ] Проверить кнопку "Continue Shopping" (возврат в каталог).
- [ ] Проверить переход к оформлению заказа ("Proceed to checkout").

### Оформление и Финализация
- [ ] Проверить корректность выбора способа оплаты.
- [ ] Проверить финальный расчет суммы заказа.
- [ ] Проверить отправку заказа ("Place order").
- [ ] Проверить страницу "Thank you":
    - [ ] Наличие текста "Your order has been received".
    - [ ] Отображение данных заказа (номер, дата, статус, сумма).
    - [ ] Отображение выбранного способа оплаты.
